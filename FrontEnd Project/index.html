CSS Transitions

Transition is a way to animate a property when certain state achieved. Example, hover

div {
width: 100px;
height: 100px;
color: red;
}
div:hover {
width: 150px;
height: 100px;
color: blue;
}

Above just a pure property changes when hover, but the result is jackling. To make it feels better, we apply transition. It will smoother the transition between property with duration and timing function you specify

 

Example:

→
Transition
property: width
duration: 1s
timing-function: ease-in;

when hover, the width of the box from 131px grow to 330px

 
→
Transition
property: box-shadow
duration: .3s
timing-function: ease-in-out

when hover, the box-shadow will come out

→
Transition
property: background, color
duration: background - 0.3s , color - 0.2s
timing-function: background - ease-in-out, color - cubic-bezier(.17,.67,.83,.67)
*border can be set on the inital state, so no need to animate it

when hover, many properties change

*note: If you want to play more regarding cubic-beizer, here is the link:https://cubic-bezier.com/#1,.16,.07,.7
